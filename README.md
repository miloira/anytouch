# AnyTouch - 远程触摸板

把手机变成电脑的无线触摸板。手机浏览器打开网页即可控制电脑鼠标，无需安装任何 App。

## 功能

- **单指滑动** — 移动鼠标光标
- **单指轻点** — 左键单击
- **双击轻点** — 双击（可最大化窗口等）
- **双击后拖动** — 拖动模式（选中文件、拖拽窗口等）
- **长按 500ms** — 进入拖动模式
- **双指上下滑动** — 垂直滚动
- **双指左右滑动** — 水平滚动
- **快速双指拉动** — 滚动加速
- **左键 / 右键按钮** — 底部按钮区，带震动反馈和按压效果
- **滚动防误触** — 双指滚动期间自动屏蔽按键
- **夜间模式** — 右上角切换，自动记忆
- **连接状态** — 触摸板顶部显示电脑名称和连接状态
- **扫码连接** — 启动后终端显示二维码，手机扫码直接打开
- **WebSocket 通信** — 低延迟实时控制

## 环境要求

- Windows 系统
- Python 3.8+
- 手机和电脑在同一局域网

## 安装

```bash
pip install websockets qrcode pywin32
```

## 使用

```bash
python anytouch.py
```

启动后终端会显示二维码和访问地址，手机扫码或浏览器输入地址即可使用。

默认端口：HTTP `8866`，WebSocket `8867`。

## 手势说明

| 手势 | 操作 |
|------|------|
| 单指滑动 | 移动光标 |
| 单指轻点 | 左键单击 |
| 快速双击 | 双击 |
| 双击 + 拖动 | 拖动 |
| 长按不动 | 进入拖动模式 |
| 双指上下滑 | 垂直滚动 |
| 双指左右滑 | 水平滚动 |
